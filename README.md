# Snake and Ladder Game in C++

## Overview
This project is a console-based Snake and Ladder Game developed using C++. The game simulates the traditional Snake and Ladder board game where players roll dice to move across the board while avoiding snakes and climbing ladders.

---

## Features
- Multiplayer Support
- Random Dice Generation
- Snake and Ladder Logic
- Turn-Based Gameplay
- Win Detection System
- Console-Based Interface

---

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Functions and Arrays

---

## Concepts Implemented
- Classes and Objects
- Loops and Conditions
- Random Number Generation
- Arrays
- Functions
